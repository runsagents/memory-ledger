# Changelog

All notable changes to this project are documented here.

## 1.0.0 — 2026-07-14

- Define the Markdown/YAML memory entry contract and JSON Schema.
- Add zero-dependency entry validation and ledger auditing for expiry, review dates, source
  provenance, scope, supersession targets, chains, and cycles.
- Enforce stronger provenance for entries marked as eligible to authorize action.
- Add four labeled synthetic examples, a deterministic generated audit report, and Node tests.
- Release the project under CC0 1.0 Universal.
